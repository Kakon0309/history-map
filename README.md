# 歴史探索マップ

国土地理院の現代地図の上に、古地図（1970年代の航空写真・明治時代の地形図）を重ねて表示するWebアプリです。

## 機能

- **古地図の切り替え**: 1970年代（航空写真）と明治時代（今昔マップ 1894-1915年）をラジオボタンで切り替え
- **透明度調整**: スライダーで古地図の透明度を変更
- **現在地表示**: 現在地を地図上にマーカー表示

## スマートフォン対応

レスポンシブデザイン対応。PWAとしてホーム画面に追加することもできます。

## 技術

- [Leaflet](https://leafletjs.com/)
- [国土地理院タイル](https://maps.gsi.go.jp/development/ichiran.html)
- [今昔マップ on the web](https://ktgis.net/kjmapw/)（埼玉大学 谷謙二）
