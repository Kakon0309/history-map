# 歴史探索マップ - 有料アプリ化ロードマップ

## ステップ

### 1. 利用規約の確認 ✅
利用しているタイル（国土地理院・今昔マップ）の利用規約を確認し、商用利用の可否と条件を把握する。

→ **結果**: [docs/LICENSE_RESEARCH.md](LICENSE_RESEARCH.md) に詳細を記載。国土地理院はリアルタイム読み込みで商用可。今昔マップは有料アプリ利用の可否を要確認（問い合わせ推奨）。

### 2. 競合との差別化
PWA を継続しつつ、オフライン対応や UI 改善などで競合との差別化を検討する。

### 3. フィードバック収集
ユーザーを募って無料版でフィードバックを集める。

### 4. アプリ化と申請
有料化する場合、Capacitor などでアプリ化し、App Store / Google Play に申請する。
